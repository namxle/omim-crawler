家族性内脏肌病 (VSCM) 是一种罕见的遗传性肌病假性梗阻形式，其特征是肠平滑肌细胞功能受损，导致肠道活动异常，严重腹痛，营养不良甚至死亡 ( Lehtonen et al., 2012 )。内脏肌病代表以家庭间和家庭内变异性为特征的疾病表型谱，其中受影响最严重的患者表现出产前膀胱增大，肠旋转不良，新生儿功能性胃肠道梗阻以及对全胃肠外营养 (TPN) 和导尿管插入术的慢性依赖 (总结 Wangler et al., 2014 ).
另一种形式的伴有功能性胃肠道阻塞的内脏肌病与眼外肌麻痹有关 ( 277320 ).
功能性胃肠道梗阻也与其他异常有关，例如 “梅花腹部” 综合征 ( 100100 ) 和巴雷特食管 (Mungan综合征； 611376 )。慢性肠假性梗阻也可能是神经性的 (见 609629 ).
